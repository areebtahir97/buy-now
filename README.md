BuyNow is a full-stack online marketplace built using <strong>Django</strong>.
It allows users to buy and sell items, search and filter listings, and communicate
securely through an integrated messaging system.
</p>

<hr>

<h2>Features</h2>
<ul>
  <li>User authentication (signup, login, logout)</li>
  <li>Create, edit, and manage item listings</li>
  <li>Category-based filtering</li>
  <li>Search functionality</li>
  <li>User-to-user messaging system</li>
  <li>Conversation inbox ordered by latest activity</li>
  <li>Image uploads for items</li>
  <li>Secure authorization checks</li>
  <li>Django Admin dashboard</li>
</ul>
